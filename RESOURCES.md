# Full-Stack Roadmap 2026 Resources

## Knowledge

- [MDN: Learn web development](https://developer.mozilla.org/en-US/docs/Learn_web_development)
  前端基础主线，覆盖 HTML、CSS、JavaScript、Web 标准、可访问性和挑战练习。用在 Q1 基础课。
- [MDN: CSS flexible box layout](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Flexible_box_layout)
  Flexbox 官方学习资料。用在布局、导航、工具条和一维排列问题。
- [MDN: CSS grid layout](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Grid_layout)
  Grid 官方学习资料。用在页面骨架、二维网格和响应式布局。
- [W3C WAI: Introduction to Web Accessibility](https://www.w3.org/WAI/fundamentals/accessibility-intro/)
  Web 可访问性高信任入口。用在语义 HTML、键盘操作、表单和交互控件。
- [React Docs: Learn React](https://react.dev/learn)
  React 官方学习路线。用在组件、JSX、状态、事件、Context、Hooks。
- [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html)
  TypeScript 官方手册。用在类型建模、函数类型、联合类型、泛型和配置。
- [TanStack Query React Docs](https://tanstack.com/query/latest/docs/framework/react/overview)
  服务端状态、缓存、请求状态和数据同步。用在 React 数据获取。
- [Vitest Guide](https://vitest.dev/guide/)
  前端和 Node 测试工具。用在组件、函数和 hook 测试。
- [Pro Git Book](https://git-scm.com/book/en/v2)
  Git 官方推荐书。用在分支、提交、远程协作、合并和变基。
- [Node.js Learn](https://nodejs.org/learn)
  Node.js 官方学习资料。用在后端运行时、HTTP、异步、测试和安全。
- [Python Tutorial](https://docs.python.org/3/tutorial/)
  Python 官方教程。用在后端语言备选、脚本和基础语法。
- [MDN: HTTP](https://developer.mozilla.org/en-US/docs/Web/HTTP)
  HTTP 概念、方法、状态码、头和缓存。用在 API 设计。
- [GraphQL Learn](https://graphql.org/learn/)
  GraphQL 官方学习资料。用在 GraphQL API、schema、query、mutation。
- [PostgreSQL Tutorial](https://www.postgresql.org/docs/current/tutorial.html)
  PostgreSQL 官方教程。用在关系型数据库、表、SQL、约束和查询。
- [RFC 7519: JSON Web Token](https://datatracker.ietf.org/doc/html/rfc7519)
  JWT 标准文档。用在认证 token 的结构和边界。
- [RFC 9700: Best Current Practice for OAuth 2.0 Security](https://datatracker.ietf.org/doc/html/rfc9700)
  OAuth 2.0 安全最佳当前实践。用在 OAuth 基础和安全取舍。
- [OWASP Top Ten](https://owasp.org/www-project-top-ten/)
  Web 应用安全风险清单。用在安全加固和面试表达。
- [AWS S3 User Guide](https://docs.aws.amazon.com/AmazonS3/latest/userguide/Welcome.html)
  对象存储、bucket、object、权限和静态资源。用在 Q3。
- [AWS Lambda Developer Guide](https://docs.aws.amazon.com/lambda/latest/dg/welcome.html)
  Serverless 计算。用在 API 后端、事件处理和运维边界。
- [Amazon API Gateway Developer Guide](https://docs.aws.amazon.com/apigateway/latest/developerguide/welcome.html)
  API 网关、路由、集成、认证和限流。用在云上 API 发布。
- [Amazon CloudFront Developer Guide](https://docs.aws.amazon.com/AmazonCloudFront/latest/DeveloperGuide/Introduction.html)
  CDN、origin、edge cache 和静态/动态内容分发。用在前端发布。
- [Amazon CloudWatch User Guide](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/WhatIsCloudWatch.html)
  日志、指标、告警、仪表盘和可观测性。用在 Q3 运维。
- [Terraform Docs](https://developer.hashicorp.com/terraform/docs)
  IaC 主资料。用在 provider、resource、state、plan、apply 和模块。
- [GitHub Actions Docs](https://docs.github.com/en/actions)
  CI/CD 官方资料。用在自动测试、构建、部署和密钥管理。
- [AWS Well-Architected Framework](https://docs.aws.amazon.com/wellarchitected/latest/framework/welcome.html)
  系统设计和云架构质量框架。用在 Q4 系统设计。
- [Google SRE Book](https://sre.google/sre-book/table-of-contents/)
  可用性、监控、发布工程、故障处理和运维智慧。用在系统设计和观测。
- [GitHub Pages Docs](https://docs.github.com/en/pages)
  作品集静态网站发布。用在 Q4 项目展示。
- [MDN: Finding a job](https://developer.mozilla.org/en-US/docs/Learn_web_development/Getting_started/Soft_skills/Finding_a_job)
  前端职业准备与作品集方向。用在简历和求职准备。

## Wisdom (Communities)

- [MDN Community](https://developer.mozilla.org/en-US/community/)
  适合验证 Web 标准理解、发现文档贡献和学习路径。
- [React Community](https://react.dev/community)
  适合寻找 React 生态、会议、博客和社区讨论入口。
- [GitHub Community](https://github.com/orgs/community/discussions)
  适合 GitHub Actions、Pages、协作流程和开源项目问题。
- [AWS re:Post](https://repost.aws/)
  AWS 官方社区问答。适合云服务配置、权限和部署问题。
- [Stack Overflow](https://stackoverflow.com/)
  通用工程问答社区。适合带最小复现的问题，但需要辨别答案时效。

## Gaps

- 还需要根据你的真实目标补充项目方向资源，例如电商、SaaS、博客系统、内部工具或数据看板。
- 如果你的目标是特定国家/地区求职，还需要补充当地简历和面试资源。
