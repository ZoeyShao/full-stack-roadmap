全栈术语表

先用统一词汇学习，避免同一个概念换名字后重新迷路。

Web 与前端

HTML
页面的结构和语义。浏览器根据 HTML 构建 DOM。
CSS
页面的呈现规则，包括布局、颜色、字体、间距和响应式。
JavaScript
浏览器和后端都可运行的编程语言。前端里主要负责交互、状态和网络请求。
DOM
浏览器把 HTML 解析成的对象树，JavaScript 通过 DOM 读取和修改页面。
Flexbox
一维布局工具，适合一行或一列里的对齐、分布和伸缩。
Grid
二维布局工具，适合页面骨架、卡片矩阵和复杂行列布局。
Responsive Design
响应式设计，让同一页面适配不同屏幕尺寸、输入方式和分辨率。
Accessibility
可访问性，让键盘用户、读屏用户、低视力用户等也能完成同样任务。
React Component
React UI 的可复用单元，本质是返回 JSX 的函数。
Props
父组件传给子组件的数据。props 应视为只读输入。
State
组件随交互变化而记住的数据。state 变化会触发重新渲染。
Server State
来自服务器的数据，如用户列表和订单详情。它需要缓存、刷新、失效和错误处理。

后端与数据

HTTP
客户端和服务器通信协议。核心对象是 request、response、method、status、headers、body。
API
系统暴露给其他系统或前端调用的接口合约。
REST Resource
REST 中的业务对象，如 users、projects、orders。用 URL 表示资源，用 HTTP method 表示动作。
CRUD
Create、Read、Update、Delete，常见业务数据操作四件套。
GraphQL
一种 API 查询语言和运行时。客户端声明需要的数据形状，服务端按 schema 解析。
Primary Key
表中唯一标识一行数据的字段，常见为 id。
Foreign Key
一个表引用另一个表主键的字段，用于表达关系和保护引用完整性。
Index
数据库为了加速查询维护的数据结构。能加快读，也会增加写入和存储成本。
Transaction
一组要么全部成功、要么全部失败的数据库操作。
Migration
数据库结构变化的版本化脚本。

认证、安全、云和运维

Authentication
认证，回答“你是谁”。例如用户名密码、OAuth 登录。
Authorization
授权，回答“你能做什么”。例如角色、权限、资源所有权。
Session
服务端保存的登录状态，浏览器通常用 cookie 携带 session id。
JWT
JSON Web Token，一种携带 claims 的 token 格式。可签名，但签名不等于加密。
OAuth
授权框架，用于让第三方应用在用户同意后访问受保护资源。
S3
AWS 对象存储。基本单位是 bucket、object、key、metadata。
CloudFront
AWS CDN，用 edge location 缓存并分发静态或动态内容。
Lambda
AWS Serverless 计算服务。按事件触发函数运行。
API Gateway
AWS API 入口服务，可把 HTTP 请求转发到 Lambda 等后端。
IaC
Infrastructure as Code，用代码声明基础设施，使环境可审查、可复现。
Terraform State
Terraform 用来记录真实资源和代码资源映射的状态文件。
CI/CD
持续集成和持续交付/部署。用流水线自动测试、构建、发布。
Observability
可观测性，通过日志、指标、追踪理解系统运行状态。
SLO
Service Level Objective，服务可靠性目标，如 99.9% 请求成功率。
System Design
系统设计，围绕需求、数据、流量、组件、可靠性和取舍进行架构推理。
