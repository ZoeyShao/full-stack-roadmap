Full-Stack Roadmap 2026 课程目录

从图片路线图扩展而来 | 16 节核心短课 | 每节 20-40 分钟 | 中文讲解，英文术语

这不是一张“要学很多东西”的清单，而是一条能交付项目的路径。你会从浏览器里的 HTML/CSS/JS 开始， 逐步走到 React、API、数据库、认证、云部署、CI/CD、监控、系统设计和作品集。

先从 Lecture Trace 入口 开始，再看 全年学习地图， 再把 术语表 放在旁边。每节课都有练习和回忆题。

16 节核心课

  1. 0001
    Web 是怎样工作的

    浏览器、URL、DNS、HTTP、HTML/CSS/JS 的边界。

  2. 0002
    HTML、CSS、JavaScript 基础闭环

    语义结构、样式层叠、DOM 事件和最小交互。

  3. 0003
    布局、响应式和可访问性

    Flex、Grid、移动端断点、键盘和表单可访问性。

  4. 0004
    React + TypeScript 组件模型

    JSX、props、state、事件、类型边界和组件拆分。

  5. 0005
    React 状态、表单、数据获取和测试

    本地状态、服务端状态、表单提交、TanStack Query、Vitest。

  6. 0006
    Git 分支、提交和协作

    branch、commit、merge、rebase、pull request 和提交故事。

  7. 0007
    后端运行时与 HTTP API

    Node/Python 的后端位置、请求响应、路由、中间件和错误。

  8. 0008
    CRUD API 设计

    资源、方法、状态码、验证、分页、幂等和 API 合约。

  9. 0009
    SQL 数据建模与查询

    表、主键、外键、连接、索引、事务和查询计划直觉。

  10. 0010
    认证、Session、JWT 与 OAuth

    身份、授权、cookie、token、OAuth flow 和常见安全坑。

  11. 0011
    云服务积木：S3、CloudFront、Lambda、API Gateway

    对象存储、CDN、Serverless、API 入口和权限边界。

  12. 0012
    Infrastructure as Code 与 Terraform

    provider、resource、state、plan、apply、module 和环境隔离。

  13. 0013
    CI/CD 流水线

    测试、构建、部署、环境、密钥、回滚和 GitHub Actions。

  14. 0014
    日志、指标和告警

    可观测性三件套、CloudWatch、SLO、告警噪音和故障复盘。

  15. 0015
    系统设计基础

    需求、容量、数据模型、缓存、队列、一致性和可靠性取舍。

  16. 0016
    项目、作品集和面试表达

    项目打磨、Demo 录制、作品集、简历 bullet 和面试故事。
