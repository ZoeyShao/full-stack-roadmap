组件边界

type ProjectCardProps = {
  title: string;
  status: "draft" | "active" | "archived";
  onArchive: () => void;
};

function ProjectCard({ title, status, onArchive }: ProjectCardProps) {
  return (
    <article>
      <h2>{title}</h2>
      <p>{status}</p>
      <button onClick={onArchive}>Archive</button>
    </article>
  );
}

props 是组件的公共接口。好组件不是“文件短”，而是输入、输出和责任清晰。

TypeScript 建模套路

type ApiResult<T> =
  | { ok: true; data: T }
  | { ok: false; error: string };

type User = {
  id: string;
  email: string;
  role: "admin" | "member";
};

function canDelete(user: User) {
  return user.role === "admin";
}

联合类型适合表达“几种互斥状态”。泛型适合表达“结构固定，但里面的数据类型变化”。

TanStack Query 心智模型

概念含义常见动作
queryKey缓存身份把资源和参数放进 key，如 ["projects", filter]。
queryFn如何获取数据调用 fetch/axios，失败时抛错。
mutation会改变服务器的数据创建、更新、删除后 invalidate 相关 query。
stale数据可能过期根据业务设置 staleTime，不要盲目 refetch。

前端测试优先级

优先级测什么不测什么
高用户关键流程，如提交表单显示成功。实现细节，如某个 private 函数被调用。
中复杂纯函数，如价格计算、权限判断。浏览器已经保证的行为。
低静态样式快照。脆弱、经常改的展示细节。
